
GUICtrlCreateDate

Создаёт дату в GUI.

GUICtrlCreateDate ( "text", left, top [, width [, height [, style [, exStyle]]]] )

---  Таблица параметров  ---
text
	Установка даты (формат всегда как "yyyy/mm/dd").
left
	Координата левого края. Если -1 тогда координата в соответствии с GUICoordMode.
top
	Координата верхнего края. Если -1 тогда координата в соответствии с GUICoordMode.
width
	[опционально] Ширина элемента (по умолчанию предыдущее значение).
height
	[опционально] Высота элемента (по умолчанию предыдущее значение).
style
	[опционально] Стиль элемента. Смотрите Стили элементов GUI в приложении.
	
	по умолчанию (-1) : $DTS_LONGDATEFORMAT
	принудительный стиль : $WS_TABSTOP
exStyle
	[опционально] Расширенный стиль элемента. Смотрите таблицу расширенных стилей.
	по умолчанию (-1) : WS_EX_CLIENTEDGE

---  Таблица возвращаемых значений  ---
Успех:	Возвращает идентификатор (controlID) нового элемента.
Ошибка:	Возвращает 0.

---  Примечания  ---
Для получения параметров элемента смотрите GUICtrlRead.
Чтобы установить или изменить параметры элемента смотрите GUICtrlUpdate....

Чтобы объединить стили со стилем по умолчанию используйте BitOr($GUI_SS_DEFAULT_DATE, новый стиль,...).

Для вывода форматированной даты / времени смотрите пример 3, чтобы понять, как использовать GuiCtrlSendMsg с $DTM_SETFORMAT.

Чтобы использовать параметры, указанные выше, вы должны включить в ваш скрипт #include <DateTimeConstants.au3> in your script.

По умолчанию стиль изменения размеров $GUI_DOCKHEIGHT (не изменяется высота).
