
FileSelectFolder

Отображает диалог выбора папки.

FileSelectFolder ( "dialog text", "root dir" [, flag [, "initial dir" [, hwnd]]] )

---  Таблица параметров  ---
dialog text
	Текст приветствия в диалоге.
root dir
	Корневой каталог дерева выбора в диалоге. Используйте "" что бы установить Рабочий Стол в качестве корневого каталога.
flag
	[опционально] 1 = показывать кнопку создания каталога (требуется IE6.0 или старше)
	 2 = использовать новый стиль диалога (требуется IE5.0 или старше)
	 4 = показывать поле редактирования (для ввода имени каталога)
initial dir
	[опционально] каталог, который будет выбран при отображении диалога (при условии, что он существует). По умолчанию нет ("").
hwnd
	[опционально] дескриптор окна для использования его в качестве родительского по отношению к данному диалогу.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает полный путь к выбранному каталогу.
Ошибка:	Возвращает "" (пустую строку) и устанавливает значение @error равным 1, если пользователь нажал кнопку отмена/закрыл окно.

---  Примечания  ---
Корневой каталог будет выбран при отображении диалога, если параметр initial dir указан, но сам каталог отсутствует на диске.
Если корневой каталог не существует, каталог Рабочего Стола (Desktop) станет корневым.
Установленный флаг "Create Folder Button" может потребовать Windows XP с IE6 и старше для работоспособности.
Специальные каталоги Windows (такие как "Мои Документы") могут быть заданы корневыми при использовании правильного CLSID, указанного в приложении.
UNC пути не поддерживаются. Если вы думаете что пользователи могут предпочесть файлы с UNC путем, то путь должен быть преображен в путь с буквой диска в начале.
