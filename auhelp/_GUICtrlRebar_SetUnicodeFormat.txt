
_GUICtrlRebar_SetUnicodeFormat

Sets the Unicode character format flag for the control

#Include 
_GUICtrlRebar_SetUnicodeFormat($hWnd[, $fUnicode = True])

---  Таблица параметров  ---
$hWnd
	Handle to rebar control
$fUnicode
	[опционально] Determines the character set that is used by the control:
		 True - The control will use Unicode characters
		False - The control will use ANSI characters
