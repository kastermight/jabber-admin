
#RequireAdmin

Указывает, что текущему скрипту необходимы полные администраторские права для работы.

#RequireAdmin

None.

---  Примечания  ---
Эта функция создана для того, чтобы позволить скриптам AutoIt корректно работать на машинах с включённым Windows Vista/7 User Account Control (UAC) (Однако будет работать и на Windows 2000 и Windows XP).
Для подробностей см. AutoIt и Windows Vista/7.

Поскольку эта функция запускает новый процесс, некоторые функции (например Consolewrite) не могут быть захвачены (SciTE не будет ничего показывать).
