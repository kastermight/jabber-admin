
StringMid

Извлекает часть символов в строке.

StringMid ( "string", start [, count] )

---  Таблица параметров  ---
string
	Используемая строка.
start
	Позиция начального символа (1 = позиция первого символа строки)
count
	[опционально] количество извлекаемых символов. По умолчанию до конца строки.

---  Таблица возвращаемых значений  ---
Возвращает извлечённую строку.

---  Примечания  ---
Если start выходит за пределы строки, возвращается пустая строка.  Если start в пределах строки, а количество выходит за пределы строки, то возвращается весь остаток строки.
