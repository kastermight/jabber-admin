
GUICtrlSetState

Изменяет состояние элемента.

GUICtrlSetState ( controlID, state )

---  Таблица параметров  ---
controlID
	Идентификатор (controlID) элемента возвращаемый функцией GUICtrlCreate....
state
	Смотрите Таблицу состояния ниже.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает 1.
Ошибка:	Возвращает 0.

---  Примечания  ---

	Таблица состояния
---  Таблица  ---
Состояние	Комментарий
0	Без изменений
$GUI_UNCHECKED	Снимает галочки Radio, Checkbox и ListViewItem.
$GUI_CHECKED	Устанавливает галочки Radio, Checkbox или ListViewItem.
$GUI_INDETERMINATE	Устанавливает Checkbox в третье состояние, attribute will be greyed.
$GUI_AVISTART	Начать воспроизведение элемента Avi.
$GUI_AVISTOP	Остановить воспроизведение элемента Avi.
$GUI_AVICLOSE	Остановить воспроизведение элемента Avi и освободить ресурс.
$GUI_DROPACCEPTED	Элемент имеет свойство принимать брошенный на него объект (файл или другой элемент). Смотрите примечание.
$GUI_NODROPACCEPTED	Элемент не будет принимать брошенный на него объект.
$GUI_SHOW	Показать скрытый элемент. Элемент Tabitem (вкладка) отображается выбранной.
$GUI_HIDE	Скрыть элемент (не отображается).
$GUI_ENABLE	Разблокировать элемент (сделать доступным).
$GUI_DISABLE	Отключить элемент (сделать серым, недоступным).
$GUI_FOCUS	Установить фокус на элементе (для нажатия Enter или ввода текста).
$GUI_NOFOCUS	Элемент Listview теряет фокус.
$GUI_DEFBUTTON	Элемент будет установлен как "кнопка по умолчанию" в окне. Смотрите примечание к TreeviewItems.
$GUI_EXPAND	Развернуть дочерние пункты TreeViewItem.
$GUI_ONTOP	Элемент будет иметь атрибут "поверх" для окна (z-порядок).

Величина состояния может суммироваться, как например $GUI_DISABLE + $GUI_HIDE установит элементы в состояние отключен и скрыт.

Если элемент AVI нужно скрыть с помощью $GUI_HIDE, то нужно сначала закрыть с помощью $GUI_AVICLOSE.

Состояние элемента "contextmenu" не может быть изменено.
Состояние элемента "listviewitem" может быть изменено, если элемент "listview" был создан с расширенным стилем $LVS_EX_CHECKBOXES. Состояние $GUI_FOCUS и $GUI_NOFOCUS могут использоваться определёнными listviewitem в элементе listview со стилем: $LVS_SHOWSELALWAYS.
Состояние элемента "menu" или "menuitem" не может быть скрыто.

! Важная информация для $GUI_EXPAND:  это состояние только для TreeViewItems. Если вы хотите использовать это "действие", то по крайней мере должен существовать один подпункт (TreeViewItem)!
Если вы хотите, чтобы выбрать другой элемент в TreeView, то вы можете использовать $GUI_FOCUS - родительский TreeView получает фокус и определённый пункт отмечается как выбранный.
Если вы хотите установить treeview пункт как пункт по умолчанию, который означает жирный шрифт, то используйте $GUI_DEFBUTTON, а чтобы отключить используйте другое значение, например 0. Это состояние не возвращается функцией GUICtrlGetState.

Если $GUI_DROPACCEPTED установлен, примите во внимание возможность drag&drop (перетащить и бросить). Элементы edit и input принимают пути к файлам.
Для других элементов при событии $GUI_EVENT_DROPPED, @GUI_DRAGID возвращает идентификатор (controlID) перетаскиваемого элемента (-1 если файл, @GUI_DRAGFILE содержит путь перетаскиваемого файла) и @GUI_DROPID возвращает идентификатор (controlID) элемента, который принимает файл / объект.
Только перемещение ListviewItem выполняется при drag & drop. @GUI_DRAGID возвращает идентификатор (controlID) ListView.
