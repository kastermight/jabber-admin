
_GUICtrlHeader_InsertItem

Inserts a new header item

#Include 
_GUICtrlHeader_InsertItem($hWnd, $iIndex, $sText[, $iWidth = 50[, $iAlign = 0[, $iImage = -1[, $fOnRight = False]]]])

---  Таблица параметров  ---
$hWnd
	Handle to control
$iIndex
	Index of the item after which the new item is to be inserted. The new item is inserted at the
		end of the control if index is greater than or equal to the number of items in the control. If index is zero,
		the new item is inserted at the beginning of the control.
$sText
	Item text
$iWidth
	[опционально] Item width
$iAlign
	[опционально] Text alignment:
		0 - Text is left-aligned
		1 - Text is right-aligned
		2 - Text is centered
$iImage
	[опционально] Zero based index of an image within the image list
$fOnRight
	[опционально] If True, the column image appears to the right of text

---  Таблица возвращаемых значений  ---
Успех:	The index of the new item
Ошибка:	-1
