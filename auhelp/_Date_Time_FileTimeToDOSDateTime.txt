
_Date_Time_FileTimeToDOSDateTime

Converts MS-DOS date and time values to a file time

#Include 
_Date_Time_FileTimeToDOSDateTime($pFileTime)

---  “аблица параметров  ---
$pFileTime
	Pointer to a $tagFILETIME structure containing the file time to convert to MS-DOS format

---  “аблица возвращаемых значений  ---
Returns an Array with the following format:
		[0] - MS-DOS date, packed as follows:
		 Bits 0- 4 Day of the month (1Ц31)
		 Bits 5- 8 Month (1 = January, 2 = February, and so on)
		 Bits 9-15 Year offset from 1980 (add 1980 to get actual year)
		[1] - MS-DOS time, packed as follows:
		 Bits 0- 4 Second divided by 2
		 Bits 5-10 Minute (0Ц59)
		 Bits 11-15 Hour (0Ц23 on a 24-hour clock)
