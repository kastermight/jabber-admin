
StringRegExpReplace

Заменяет текст строки используя регулярное выражение.

StringRegExpReplace ( "test", "pattern", "replace", [count] )

---  Таблица параметров  ---
test
	Строка для поиска
pattern
	Образец для поиска (шаблон). См. StringRegExp для создания шаблона.
replace
	Текст замены регулярного выражения.  Чтобы вставить текст группы используйте, \0 - \9 (или $0 - $9) как обратную ссылку.
count
	[опционально] количество замен в строке.  По умолчанию 0 - выполняет замену всех найденных образцов.

---  Таблица возвращаемых значений  ---
@Error	принимает значения
0	Успешное выполнение. @extended содержит количество выполненных замен.
2	Ошибка в регулярном выражении. @extended содержит позицию ошибочного символа в регулярном выражении.

---  Примечания  ---
Для разделения обратных ссылок от фактических (заменяемых) чисел, обрамите ссылки фигурными скобками: "${1}5".
