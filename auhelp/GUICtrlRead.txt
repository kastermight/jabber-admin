
GUICtrlRead

Возвращает состояние или данные элемента управления.

GUICtrlRead ( controlID [, advanced] )

---  Таблица параметров  ---
controlID
	Идентификатор элемента управления, возвращаемый функцией GUICtrlCreate....
advanced
	[опционально] возвращает расширенную информацию элемента управления.
	0 = (Default) Возвращает значение состояния или данные элемента управления.
	1 = Возвращает расширенную информацию элемента управления (смотрите Примечание).

---  Таблица возвращаемых значений  ---
Успех:	Возвращает взависимости от типа элемента управления (смотрите ниже).
Ошибка:	Возвращает 0.

---  Таблица  ---
Элемент	Возвращаемое значение
Checkbox, Radio	состояние элемента. See State table
Combo, List	Выбранное значение
Input, Edit	Текстовое содержимое элемента
Button	Отображаемый текст на кнопке
Date	Выбранную дату в формате определяемом региональными настройками
Progress	Текущий процент прогресса
Slider	Текущую величину
Tab	Номер или идентификатор элемента управления вкладки (tabitem) взависимости от значения расширенных параметров.
Menu, MenuItem	Состояние меню / элементов меню. Смотрите таблицу состояния элементов
TreeView	Идентификатор элемента управления (controlID) выбранного TreeViewItem
TreeViewItem	Состояние TreeViewItem
ListView	Идентификатор элемента управления (controlID) выбранного ListViewItem. Или 0, если ничего не выбрано.
Dummy	Величина установленная GUICtrlSendToDummy или GUICtrlSetData

---  Примечания  ---
В расширенном режиме возвращается величина содержащая дополнительные данные элемента управления (смотрите ниже).

Примечание: Не для всех элементов доступны расширенные параметры!

---  Таблица  ---
Элемент	Возвращаемое значение (расширенные)
Checkbox, Radio	Отображаемый текст на элементе управления.
Menu, MenuItem	Отображаемый текст на элементе управления.
TreeView	Текст текущего выбранного TreeViewItem.
TreeViewItem	Текст TreeViewItem.
ListViewItem	Состояние ListViewItem, если используется доп. стиль $LVS_EX_CHECKBOXES.  Смотрите таблицу состояния элементов
Tab	идентификатор (controlID) выбранной вкладки (tabitem).

Для элементов Checkbox, Radio устанавливается несколько состояний и могут возвращаться $GUI_FOCUS и $GUI_CHECKED. Используйте BitAnd(GUICtrlRead($Item),$GUI_CHECKED) для проверки установленной галочки.

Для пунктов элемента Listview устанавливается несколько состояний и могут возвращаться $GUI_CHECKED и $GUI_UNCHECKED (только для listview со стилем LVS_EX_CHECKBOXES и включенным расширенным параметром) . Используйте BitAnd(GUICtrlRead($Item),$GUI_CHECKED) для проверки установленной галочки.

Для пунктов элементов  Treeview устанавливается несколько состояний и могут возвращаться $GUI_FOCUS, $GUI_EXPAND и $GUI_CHECKED, $GUI_UNCHECKED (только для treeview со стилем TVS_CHECKBOXES). Используйте BitAnd(GUICtrlRead($Item),$GUI_CHECKED) для проверки установленной галочки.
