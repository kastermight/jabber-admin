
_GUICtrlComboBox_GetExtendedUI

Determines whether a ComboBox has the default user interface or the extended user interface

#Include 
_GUICtrlComboBox_GetExtendedUI($hWnd)

---  Таблица параметров  ---
$hWnd
	Handle to control

---  Таблица возвращаемых значений  ---
True: ComboBox has the extended user interface
False: ComboBox does "NOT" have the extended user interface

---  Примечания  ---
By default, the F4 key opens or closes the list and the DOWN ARROW changes the current selection.

In a ComboBox with the extended user interface, the F4 key is disabled and pressing the DOWN ARROW
key opens the drop-down list
