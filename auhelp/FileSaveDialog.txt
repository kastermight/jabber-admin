
FileSaveDialog

Отображает диалог сохранения файла.

FileSaveDialog ( "title", "init dir", "filter" [, options [, "default name" [, hwnd]]] )

---  Таблица параметров  ---
title
	Заголовок диалогового окна.
init dir
	Каталог, который будет открыт изначально в диалоговом окне.
filter
	Типы файлов, доступные для выбора. Например, "Все (*.*)", "Текстовые файлы (*.txt)" или "Все (*.*)|Текстовые файлы (*.txt)" (см. примечания).
options
	[опционально]  опции диалога: сложите необходимые значения, чтобы использовать опции вместе.
	2 = путь должен существовать (если пользователем был выбран путь, оканчивающийся обратным слэшем)
	 16 = выдать запрос на перезапись файла
default name
	[опционально] имя файла, которое будет выбрано при создании диалога. По умолчанию нет ("").
hwnd
	[опционально] дескриптор окна для использования в качестве родительского по отношению к этому диалоговому окну.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает полный путь и имя выбранных файлов. Результат для нескольких объектов: "Каталог|Файл1|Файл2|..."
Ошибка:	Устанавливает значение @error
@error:	1 - Ошибка при выборе файла.
	2 - Неправильный фильтр.

---  Примечания  ---
Разделяйте фильтры выбора типов файлов с помощью точки с запятой, как указано в примере.
Если необходимо использовать несколько групп фильтров, следует разделять их с помощью символа "|".

Если указано имя файла по умолчанию, опции диалога также должны быть указаны. Если нет необходимости их использовать, оставьте значение параметра равным 0.

Некоторые специальные каталоги Windows (например, "Мои документы") могут быть установлены в качестве начального каталога, смотрите в приложении.

@WorkingDir изменяется при удачном выполнении функции.
