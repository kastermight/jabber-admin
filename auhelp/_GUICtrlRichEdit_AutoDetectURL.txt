
_GUICtrlRichEdit_AutoDetectURL

Включение или отключение автоматического определения гиперссылок интернета (URL)

#Include 
_GUICtrlRichEdit_AutoDetectURL($hWnd, $fState)

---  Таблица параметров  ---
$hWnd
	Дескриптор элемента
$fState
	True для обнаружения всех ссылок в тексте, False для отключения обнаружения

---  Таблица возвращаемых значений  ---
Успех:	Возвращает True
Ошибка:	Возвращает False и устанавливает @error
@error:	101 - $HWnd не является дескриптором
	102 - $fState не является True или False
	700 - внутренняя ошибка, например, недостаточно памяти

---  Примечания  ---
Если состояние установлено, то любая модификация текста приводит к проверке текста на соответствие URL-формату. Распознаются строки начинающиеся со следующих URL-адресов, таких как: http:, file:, mailto:, ftp:, https:, gopher:, nntp:, prospero:,telnet:, news:, wais:. При обнаружении URL, Windows устанавливает атрибуты ссылки для всех символов URL-адреса, и подчеркивает строку.

При автоматическом обнаружении URL и URL найден, то Windows удаляет атрибуты ссылок со всех символов, которые не являются URL-адресом.

Для уведомительных сообщений, используйте функцию _GUICtrlRichEdit_SetEventMask с $ENM_LINK
