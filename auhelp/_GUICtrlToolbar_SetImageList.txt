
_GUICtrlToolbar_SetImageList

Sets the default button image list

#Include 
_GUICtrlToolbar_SetImageList($hWnd, $hImageList)

---  Таблица параметров  ---
$hWnd
	Handle to the control
$hImageList
	Handle to the image list to set. If this parameter is 0, no images are displayed in the buttons.

---  Таблица возвращаемых значений  ---
Успех: The handle to the image list previously used to display buttons in their default state
Ошибка: 0
