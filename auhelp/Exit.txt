
Exit

Завершает выполнение скрипта.

Exit [return code]

---  Таблица параметров  ---
return code
	[опционально] Целое число, являющееся кодом выхода. Этот код можно получить в Windows или DOS, используя переменную %ERRORLEVEL%. По умолчанию равно 0. Скрипт устанавливает errorlevel на 0, если работает нормально; error level 1 и выше обычно означает, что скрипт работает не так, как надо.

---  Примечания  ---
Параметр, если указан, может быть указан в скобках. Следующие вызовы одинаковы: Exit, Exit 0, и Exit(0). Однако, Exit() неправильно.

Этот код может быть получен при использовании функции OnAutoItExitRegister с помощью @EXITCODE.
