
_GUICtrlRichEdit_GetTextLength

Возвращает количество всех символов в тексте элемента

#Include 
_GUICtrlRichEdit_GetTextLength($hWnd [, $fExact = True [, $fChars = False]])

---  Таблица параметров  ---
$hWnd
	Дескриптор элемента
$fExact
	[опционально] True возвращает точное количество
		       - = False возвращает по крайней мере количество символов в элементе (быстрее)
		По умолчанию: точное количество
$fChars
	[опционально] True - возвращает количество измеряемое в символах
		          = False - возвращает количество измеряемое в байтах
		По умолчанию: в байтах

---  Таблица возвращаемых значений  ---
Успех:	Возвращает количество в байтах или символах
Ошибка:	Возвращает 0 и устанавливает @error
@error:	101 - $HWnd не является дескриптором
	102 - $fExact не является True или False
	103 - $fChars не является True или False
