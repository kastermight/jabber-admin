
GUICtrlCreateRadio

Создаёт Radio-кнопку (переключатель) в GUI.

GUICtrlCreateRadio ( "text", left, top [, width [, height [, style [, exStyle]]]] )

---  Таблица параметров  ---
text
	Текст элемента
left
	Координата левого края. Если -1 тогда координата в соответствии с GUICoordMode.
top
	Координата верхнего края. Если -1 тогда координата в соответствии с GUICoordMode.
width
	[опционально] Ширина элемента (по умолчанию устанавливается по длине текста).
height
	[опционально] Высота элемента (по умолчанию устанавливается по высоте текста).
style
	[опционально] Стиль элемента. Смотрите Стили элементов GUI в приложении.
	
	по умолчанию (-1) : отсутствует.
	принудительный стиль : $BS_AUTORADIOBUTTON и $WS_TABSTOP если первая радиокнопка в группе.
exStyle
	[опционально] Расширенный стиль элемента. Смотрите таблицу расширенных стилей.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает идентификатор (controlID) нового элемента.
Ошибка:	Возвращает 0.

---  Примечания  ---
Для получения параметров элемента смотрите GUICtrlRead.
Чтобы установить или изменить параметры элемента смотрите GUICtrlUpdate....

Чтобы объединить стили со стилем по умолчанию используйте BitOr($GUI_SS_DEFAULT_RADIO, новый стиль,...).

Чтобы использовать параметры, указанные выше, вы должны включить в ваш скрипт #include <ButtonConstants.au3>.

По умолчанию стиль изменения размеров $GUI_DOCKHEIGHT (не изменяется высота).
