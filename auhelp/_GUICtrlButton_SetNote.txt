
_GUICtrlButton_SetNote

Sets the text of the note associated with a command link button

#Include 
_GUICtrlButton_SetNote($hWnd, $sNote)

---  Таблица параметров  ---
$hWnd
	Handle to the control
$sNote
	String that contains the note

Успех: True
Ошибка: False
