
_WinNet_RestoreConnection

Restores the connection to a network resource

#Include 
_WinNet_RestoreConnection([$sDevice = "" [, $hWnd = 0 [, $fUseUI = True]]])

---  Таблица параметров  ---
$sDevice
	[опционально] The local name of the drive to connect to, such as "Z:". If blank, the function reconnects all
		persistent drives stored in the registry for the current user.
$hWnd
	[опционально] Handle to the parent window that the function uses to display the user interface that prompts
		the user for a name and password when making the network connection. If 0, there is no owner window.
$fUseUI
	[опционально] If True, display a username/password prompt to the caller

---  Таблица возвращаемых значений  ---
Успех:	True
Ошибка:	False
