
ToolTip

Создаёт всплывающую подсказку в любом месте экрана.

ToolTip ( "text" [, x [, y [, "title" [, icon [, options]]]]] )

---  Таблица параметров  ---
text
	Текст всплывающей подсказки. Пустая строка означает, что отображаемая всплывающая подсказка будет очищена.
x
	[опционально] координата всплывающей подсказки по x. По умолчанию используется текущая координата курсора мыши.
y
	[опционально] координата всплывающей подсказки по y. По умолчанию используется текущая координата курсора мыши.
title
	[опционально] заголовок всплывающей подсказки (Требует IE5+)
icon
	[опционально] иконка, отображаемая рядом с заголовком: требует указания параметра title.
		0 = без иконки, 1 = иконка "информация", 2 = иконка "предупреждение", 3 = иконка "ошибка"
options
	[опционально] дополнительные опции (сложите необходимые значения):
		1 = отобразить, как Balloon Tip (Требует IE5+)
		2 = центрировать подсказку по координатам x,y (по умолчанию это координаты левого верхнего угла подсказки).
		4 = принудительно отображать подсказку полностью, независимо от указанных координат. Если используется несколько мониторов, подсказка будет "прилипать" к ближайшему монитору.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает 1.
Ошибка:	Возвращает 0, если длина заголовка превышает 99 символов.

---  Примечания  ---
Чтобы пропустить указание опционального параметра, оставьте вместо него значение по умолчанию, равное Default.

Всплывающая подсказка будет отображаться до завершения скрипта или до вызова ToolTip("").
Можно использовать символы @CR или @LF для создания многострочной всплывающей подсказки.

Если используются опции 1 (Balloon Tip) и 2 (центрирование по координатам) одновременно, stem будет находиться над центром всплывающей подсказки в указанных координатах.
