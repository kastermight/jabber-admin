
Eval

Возвращает значение переменной с указанным именем.

Eval ( string )

---  Таблица параметров  ---
string
	Имя переменной

---  Таблица возвращаемых значений  ---
Успех:	Возвращает значение переменной.
Ошибка:	Возвращает "" (пустую строку) и устанавливает значение @error не равным 0.

---  Примечания  ---
Если есть необходимость в использовании Assign() для создания или присвоения значения переменной, то, в таких ситуациях, Eval() используется для чтения значения переменной, а IsDeclared() используется для проверки её существования.
