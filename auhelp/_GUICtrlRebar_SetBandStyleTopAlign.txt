
_GUICtrlRebar_SetBandStyleTopAlign

Set whether to keep band in top row

#Include 
_GUICtrlRebar_SetBandStyleTopAlign($hWnd, $iIndex[, $fEnabled = True])

---  Таблица параметров  ---
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
$fEnabled
	[опционально] If True the item state is set, otherwise it is not set

---  Таблица возвращаемых значений  ---
Успех:	True
Ошибка:	False
