
_GUICtrlMenu_SetItemEnabled

Sets the enabled state of a menu item

#Include 
_GUICtrlMenu_SetItemEnabled($hMenu, $iItem[, $fState = True[, $fByPos = True]])

---  Таблица параметров  ---
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$fState
	[опционально] Item state to set:
		 True - State is enabled
		False - State is disabled
$fByPos
	[опционально] Menu identifier flag:
		 True - $iItem is a zero based item position
		False - $iItem is a menu item identifier

---  Таблица возвращаемых значений  ---
Успех:	True
Ошибка:	False
