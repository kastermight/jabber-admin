
_GUICtrlEdit_ShowBalloonTip

Отображает всплывающую подсказку, связанную с окном редактирования

#Include 
_GUICtrlEdit_ShowBalloonTip($hWnd, $sTitle, $sText, $iIcon)

---  Таблица параметров  ---
$hWnd
	Дескриптор элемента
$sTitle
	Заголовок всплывающей подсказки (Unicode)
$sText
	Текст всплывающей подсказки (Unicode)
$iIcon
	Иконка может быть одним из следующих значений:
		$TTI_ERROR - Использовать иконку ошибки
		$TTI_INFO - Использовать иконку информации
		$TTI_NONE - Не использовать иконку
		$TTI_WARNING - Использовать иконку предупреждения
		Следующие значения для OS Vista и выше
		$TTI_ERROR_LARGE - Использовать иконку ошибки
		$TTI_INFO_LARGE - Использовать иконку информации
		$TTI_WARNING_LARGE - Использовать иконку предупреждения

---  Таблица возвращаемых значений  ---
Успех:	Возвращает True
Ошибка:	Возвращает False

---  Примечания  ---
Минимальная версия OS - Windows XP.
