
SplashTextOn

Создаёт настроенное всплывающее окно с текстом.

SplashTextOn ( "title", "text" [, w [, h [, x pos [, y pos [, opt [, "fontname" [, fontsz [, fontwt]]]]]]]] )

---  Таблица параметров  ---
title
	Заголовок окна всплывающего окна.
text
	Текст всплывающего окна.
w
	опционально] ширина окна в пикселях. (по умолчанию равна 500)
h
	[опционально] высота окна в пикселях. (по умолчанию равна 400)
x pos
	[опционально] отступ слева (в пикселях) для окна с изображением (по умолчанию окно центрируется).
y pos
	[опционально] отступ сверху (в пикселях) для окна с изображением (по умолчанию окно центрируется).
opt
	[опционально] по умолчанию 'всегда поверх всех окон/с заголовком/выравнивание по центру'
	Сложите необходимые значения:
	 0 = выравнивание по центру/всегда поверх всех окон/с заголовком (по умолчанию)
	 1 = окно без заголовка и границ
	 2 = без атрибута "всегда поверх всех окон"
	 4 = выравнивание текста по левому краю
	 8 = выравнивание текста по правому краю
	16 = окно может быть перемещено
	32 = центрировать текст по вертикали
fontname
	[опционально] шрифт для использования (если значение "" или шрифт не найден, будет использован шрифт ОС по умолчанию для GUI)
fontsz
	[опционально] размер шрифта (по умолчанию 12; стандартные размеры: 6 8 9 10 11 12 14 16 18 20 22 24 26 28 36 48 72)
fontwt
	[опционально] Font weight (0 - 1000, default = 400 = normal). Значение > 1000 интерпретируется, как 0.

---  Таблица возвращаемых значений  ---
Возвращает дескриптор всплывающего окна, который может быть использован в функции ControlSetText.

---  Примечания  ---
Чтобы пропустить указание опционального параметра, оставьте вместо него значение по умолчанию, равное:
  "" для строковых параметров
  -1 для числовых параметров

Только одно окно SplashImage/Text может быть отображено одновременно.

Even better is to use ControlSetText to update text without flicker...
If the text is center and multiline, the ControlSetText will not override the number of lines created by the SplashTextOn.

Всплывающее окно с параметром opt равным 1 не может быть перемещено и не может быть активировано.

Имена стандартных шрифтов:
    Arial, Comic Sans MS, Courier New, Lucida Console, Microsoft Sans Serif, System, Tahoma, Times New Roman, и WingDings
См. Приложения для получения полного списка шрифтов

Используйте @LF для отображения нескольких строк текста.
