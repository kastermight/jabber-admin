
_ArrayMin

Возвращает наименьшее значение содержащееся в массиве.

#Include 
_ArrayMin(Const ByRef $avArray [, $iCompNumeric = 0 [, $iStart = 0 [, $iEnd = 0]]])

---  Таблица параметров  ---
$avArray
	Массив для поиска наибольшего значения
$iCompNumeric
	[опционально] Метод сравнения:
		0 - сравнить по алфавиту
		1 - сравнить по числам
$iStart
	[опционально] Индекс массива, от которого начинать поиск
$iEnd
	[опционально] Индекс массива, в котором закончить поиск

---  Таблица возвращаемых значений  ---
Успех:	Возвращает наименьшее значение в массиве
Ошибка:	Возвращает "" (пустую строку), устанавливает @error (смотрите описание кодов ошибок функции _ArrayMinIndex() )
