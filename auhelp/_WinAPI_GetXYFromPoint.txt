
_WinAPI_GetXYFromPoint

#Include

---  Таблица параметров  ---
$tPoint
	$tagPOINT structure
$iX
	Returns X value
$iY
	Returns Y value

---  Примечания  ---
This function extracts the X/Y values from a $tagPOINT structure
