
OnAutoItExitRegister

Регистрирует функцию для её выполнения при завершении работы AutoIt.

OnAutoItExitRegister ( "function" )

---  Таблица параметров  ---
function
	Имя пользовательской функции для вызова.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает 1. @extended устанавливается равным 1, если функция с этим именем уже зарегистрирована.
Ошибка:	Возвращает 0.

---  Примечания  ---
Код выхода может быть получен с помощью @ExitCode.
Метод выхода может быть получен с помощью @ExitMethod.
---  Таблица параметров  ---
0
	Обычное завершение.
1
	Выход с помощью функции Exit.
2
	Выход с помощью меню значка в системном трее.
3
	Завершение работы скрипта в связи с выходом пользователя из системы (user logoff).
4
	Выход в связи с завершением работы Windows.

Не используйте одну и ту же функцию для события $GUI_EVENT_CLOSE, это приведёт к повторному выполнению функции. Данная функция выполняется даже при завершении скрипта с ошибкой, что позволяет сохранить данные, освободить ресурсы (GUIDelete() для RichEdit, или функции с суффиксом Destroy)
