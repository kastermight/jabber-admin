
GUISetFont

Устанавливает шрифт по умолчанию для GUI-окна.

GUISetFont ( size [, weight [, attribute [, fontname [, winhandle [, quality]]]]] )

---  Таблица параметров  ---
size
	Размер шрифта (по умолчанию 8.5).
weight
	[опционально] Толщина шрифта (по умолчанию 400 = нормальный) (700 - жирный).
attribute
	[опционально] Стиль: 2 - курсив, 4 - подчёркнутый, 8 - зачёркнутый (объединение стилей: 2+4 = курсив и подчёркнутый).
fontname
	[опционально] Имя шрифта. (По умолчанию используется шрифт OS, если не указан "" или не найден).
winhandle
	[опционально] Дескриптор окна возвращаемый функцией GUICreate (по умолчанию предыдущее используемое окно).
quality
	[опционально] Сглаживание шрифта (по умолчанию PROOF_QUALITY=2).

---  Таблица возвращаемых значений  ---
Успех:	Возвращает 1.
Ошибка:	Возвращает 0.

---  Примечания  ---
Смотрите Приложение для получения полного списка шрифтов Windows и версий Windows, в которых они поддерживаются.

Размер шрифта может содержать десятичные доли, например 8.5.

Для некоторых элементов, как например Label, по умолчанию может быть 8.5 вместо 9 в соответствии с величинами темы Windows.

Для параметров Сглаживания проверьте MSDN, некоторые установки windows XP требуют CLEARTYPE_QUALITY=5
