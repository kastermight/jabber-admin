
GUICtrlRegisterListViewSort

Регистрация пользовательской функции для внутренней сортировки ListView функцией обратного вызова.

GUICtrlRegisterListViewSort ( controlID, "function" )

---  Таблица параметров  ---
controlID
	Идентификатор ListView, для которого должна выполнится функция.
function
	Имя пользовательской функции, вызываемой, когда выполняется сортировка обратным вызовом.

---  Таблица возвращаемых значений  ---
Успех:	1
Ошибка:	0

---  Примечания  ---
!!! Чтобы функция была работоспособной, нужно определить максимум 4 параметра функции иначе функция не будет вызываться !!!
i.e:
Func MySortFunction($nListViewID, $LParam1, $LParam2, $nColumn)
...
EndFunc

Or

Func MySortFunction($nListViewID, $LParam1, $LParam2)
...
EndFunc

Когда функция вызывается, то 4 параметра принимают следующие значения:
---  Таблица  ---
Позиция	Параметр	Описание
1	controlID	Идентификатор ListView для которой используется возврат функции.
2	lParam1	lParam значение первого пункта (by default the item controlID).
3	lParam2	lParam значение второго пункта (by default the item controlID).
4	column	колонка, которая была нажата для сортировки (Номер первый колонки - 0).

Следующие значения должны быть Returned изменив поведение сортировки обратным вызовом:
---  Таблица  ---
Возвращаемое значение	Описание
-1	1-ый пункт должен предшествовать 2-му.
0	Без изменений.
1	1-ый пункт должен последовать за 2-ым.
