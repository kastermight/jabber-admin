
_WinAPI_GetMousePosX

Returns the current mouse X position

#Include 
_WinAPI_GetMousePosX([$fToClient = False [, $hWnd = 0]])

---  Таблица параметров  ---
$fToClient
	[опционально] If True, the coordinates will be converted to client coordinates
$hWnd
	[опционально] Window handle used to convert coordinates if $fToClient is True

---  Таблица возвращаемых значений  ---
Успех:	Mouse X position
Ошибка:	set @error

---  Примечания  ---
This function takes into account the current MouseCoordMode setting when obtaining the mouse position. It
will also convert screen to client coordinates based on the parameters passed.
