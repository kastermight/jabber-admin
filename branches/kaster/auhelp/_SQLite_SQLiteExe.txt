
_SQLite_SQLiteExe

Executes commands in SQLite.exe

#include 
_SQLite_SQLiteExe ( $sDatabaseFile, $sInput, ByRef $sOutput [, $sSQLiteExeFilename = "SQLite3.exe" ] )

---  Таблица параметров  ---
$sDatabaseFile
	Database filename
$sInput
	Commands for SQLite.exe
$sOutput
	Raw output from SQLite.exe
$sSQLiteExeFilename
	[опционально] Path to SQlite.exe

---  Таблица возвращаемых значений  ---
Успех:	Returns $SQLITE_OK
Ошибка:	Return value can be compared against $SQLITE_* constants
@error:	1 - Can't create new Database
	2 - SQLite3.exe not Found
	3 - SQL error / incomplete SQL
 	4 - Can't open input file

---  Примечания  ---
Run SQLite.exe, press enter and type .Help for more info about the SQLite3.exe Commands.

SQLite3.exe can be found at the SQLite Web site. A copy with an help file can be found in AutoIt3\Extras\SQLite dir.
