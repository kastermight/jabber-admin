
SetExtended

Вручную устанавливает значение @extended.

SetExtended ( code [, return value] )

---  Таблица параметров  ---
code
	Значение (целое) устанавливаемое в @extended.
return value
	[опционально] установить значение возвращаемое функцией, используя Return.

---  Таблица возвращаемых значений  ---
По умолчанию ничего не возвращает, однако, если указан дополнительный параметр, функция возвращает его.

---  Примечания  ---
Вначале выполнения функции @extended равен 0.  Если вызвана SetExtended() перед функцией, то @extended становится равной 0 после завершения функции.  Это означает, что для того, чтобы @extended установить после выполнения функции, вы должны явно установить значение.   Это также означает, что вам возможно потребуется продублировать значение @extended в переменную, если вы проверяете его в цикле While-WEnd.
