
_GUICtrlRichEdit_StreamToFile

Записать содержимое элемента Rich Edit в файл

#Include 
_GUICtrlRichEdit_StreamToFile($hWnd, $sFilespec [, $fIncludeCOM=True [, $iOpts=0 [, $iCodePage = 0]]])

---  Таблица параметров  ---
$hWnd
	Дескриптор элемента
$sFileSpec
	Путь к файлу
$fIncludeCOM
	[опционально] True (по умолчанию):
		    Если записывается в файл .rtf, то включает в себя любые COM-объекты (space consuming)
		    Если записывается в любой другой файл, то записывается текстовое представление COM-объектов
		False: Записывает пробелы вместо COM-объектов
$iOpts
	[опционально] Дополнительные опции: (по умолчанию: 0)
		$SFF_PLAINTRTF - записывать только rich text keywords общий для всех языков
		$SF_UNICODE - записывать Unicode
$iCodePage
	[опционально] Создать UTF-8 и текст использующий эту кодовую страницу
		По умолчанию: не использовать

---  Таблица возвращаемых значений  ---
Успех:	Возвращает True
Ошибка:	False и устанавливает @error
@error:	101 - $HWnd не является дескриптором
	102 - Невозможно создать $sFilespec
	1041 - $SFF_PLAINRTF недопустим для текстового файла
	1042 - $opts: недопустимые опции
	1043 - $SF_UNICODE допустимо только для текстовых файлов
	700 - внутренняя ошибка

---  Примечания  ---
Если текст выделен, то заменяется выделенное, иначе заменяется весь текст в элементе Rich Edit

Если расширение файла в $sFileSpec является .rtf, то записывается RTF, иначе текст
