
BinaryMid

Возвращает часть байтов из бинарного варианта.

BinaryMid ( binary, start [, count] )

---  Таблица параметров  ---
binary
	Бинарные данные.
start
	Позиция начала (1 = первый байт)
count
	[опционально] необходимое число байтов. По умолчанию возвращается оставшаяся часть бинарных данных.

---  Таблица возвращаемых значений  ---
Возвращает часть байтов из бинарного варианта.

---  Примечания  ---
Если параметр start задан неправильно, будет возвращён пустой бинарный вариант. Если параметр start задан правильно, а count нет, будет возвращена оставшаяся часть бинарной переменной.
