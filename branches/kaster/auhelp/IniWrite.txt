
IniWrite

Записывает значение параметра в стандартный .ini файл.

IniWrite ( "filename", "section", "key", "value" )

---  Таблица параметров  ---
filename
	Имя .ini файла.
section
	Имя раздела в .ini файле.
key
	Имя параметра в .ini файле.
value
	Значение параметра для записи.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает 1.
Ошибка:	Возвращает 0, если файл не доступен для записи.

---  Примечания  ---
Стандартный .ini файл имеет следующую структуру:
[Имя раздела]
Параметр=Значение

Если файл не существует, то он будет создан. Однако, любые несуществующие каталоги не будут создаваться. Параметры и разделы добавляются в конце (без какой-либо сортировки).
При записи значения, "обрамлённого" кавычками, они удаляются. Чтобы записать кавычки в значение, необходимо дублировать их. Например, чтобы записать "Это тест" в качестве значения параметра, необходимо использовать ""Это тест"".
Начальные и конечные пробелы удаляются. Чтобы записать их в значение, необходимо обрамлять записываемую строку кавычками. Например, при использовании , "    это тест с пробелами " они (пробелы) сохранятся в значении (а кавычки будут удалены).
Многострочные значения не поддерживаются.
