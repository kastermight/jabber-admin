
_Crypt_DecryptFile

Расшифровывает файл с указанными ключом и алгоритмом

#Include 
_Crypt_DecryptFile($sSourceFile, $sDestinationFile, $vCryptKey, $iALG_ID)

---  Таблица параметров  ---
$sSourceFile
	Файл для расшифровки
$sDestinationFile
	Файл для сохранения расшифрованного файла
$vCryptKey
	Пароль или указатель ключа, если указан флаг CALG_USERKEY
$iALG_ID
	Алгоритм дешифрования

---  Таблица возвращаемых значений  ---
Успех:	Возвращает True
	Устанавливает @error = 0
Ошибка:	Возвращает -1 и устанавливает @error:
	1 - Не удалось создать ключ
	2 - Не удалось открыть файл  для обработки
	3 - Не удалось открыть файл для сохранения
	4 - Не удалось расшифровать заключительную часть
	5 - Не удалось расшифровать часть
