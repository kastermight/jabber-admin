
MsgBox

Отображает простое окно сообщения с опциональным временем отображения.

MsgBox ( flag, "title", "text" [, timeout [, hwnd]] )

---  Таблица параметров  ---
flag
	Параметр указывает на тип окна сообщения и возможные комбинации отображаемых кнопок. См. примечания.
title
	Заголовок окна сообщения.
text
	Текст окна сообщения.
timeout
	[опционально] время отображения в секундах. По истечении указанного промежутка времени окно сообщения будет автоматически закрыто. Значение по умолчанию равно 0, что означает, что время отображения окна сообщения не ограничено.
hwnd
	[опционально] дескриптор окна (window handle) для использования в качестве родительского по отношению к этому диалогу.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает ID нажатой кнопки.
Ошибка:	Возвращает -1, если истекло время отображения окна сообщения.

---  Примечания  ---
Параметр flag может быть комбинацией следующих значений:

---  Таблица  ---
десятичное значение параметра flag	Отображаемые кнопки	шестнадцатеричное значение параметра flag
0	Кнопка OK	0x0
1	Кнопки OK и Отмена	0x1
2	Кнопки Прервать, Повтор и Пропустить	0x2
3	Кнопки Да, Нет и Отмена	0x3
4	Кнопки Да и Нет	0x4
5	Кнопки Повтор и Отмена	0x5
6 **	Кнопки Отмена, Повторить, Продолжить	0x6
десятичное значение параметра flag	Отображаемая иконка	шестнадцатеричное значение параметра flag
0	(Нет)	0x0
16	Иконка "стоп"	0x10
32	Иконка "знак вопроса"	0x20
48	Иконка "предупреждение"	0x30
64	Иконка "информация", с 'i' в кружке	0x40
десятичное значение параметра flag	Выделенная кнопка по умолчанию	шестнадцатеричное значение параметра flag
0	Первая кнопка	0x0
256	Вторая кнопка	0x100
512	Третья кнопка	0x200
десятичное значение параметра flag	Модальность	шестнадцатеричное значение параметра flag
0	Приложение (Application)	0x0
4096	Системная, поверх всех окон (System modal)	0x1000
8192	Блокирует родительское окно (Task modal)	0x2000
десятичное значение параметра flag	Другое	шестнадцатеричное значение параметра flag
0	(ничего специального)	0x0
262144	MsgBox имеет атрибут "поверх всех окон"	0x40000
524288	Заголовок и текст выравниваются по правому краю	0x80000

** Доступно только в Windows 2000/XP и выше.

К примеру, чтобы указать окно SYSTEMMODAL с кнопками Да/Нет, следует указать параметр flag равным 4096+4 (или 4100). При использовании шестнадцатеричного значения - 0x1000+0x4 (или 0x1004).

Окно сообщения появляется в центре экрана с размерами, пропорциональными выводимому тексту. Однако, заголовок может быть обрезан при использовании флага SYSTEMMODAL (4096).

Если параметр "title" равен ключевому слову Default, будет использован макрос @ScriptName
