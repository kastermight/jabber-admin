
_WinAPI_ReadFile

Reads data from a file

#Include 
_WinAPI_ReadFile($hFile, $pBuffer, $iToRead, ByRef $iRead [, $pOverlapped = 0])

---  Таблица параметров  ---
$hFile
	Handle to the file to be read
$pBuffer
	Pointer to the buffer that receives the data read from a file
$iToRead
	Maximum number of bytes to read
$iRead
	Number of bytes read
$pOverlapped
	[опционально] Pointer to a $tagOVERLAPPED structure

---  Таблица возвращаемых значений  ---
Успех:	True
Ошибка:	False
