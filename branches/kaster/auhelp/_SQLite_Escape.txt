
_SQLite_Escape

Escapes a string

#include 
_SQLite_Escape ( $sString [, $iBuffSize ] )

---  Таблица параметров  ---
$sString
	string to escape
$iBuffSize 
	[опционально] the result will be truncated if its longer

---  Таблица возвращаемых значений  ---
Успех:	Returns escaped string
Ошибка:	Returns empty string and sets @error
@error:	1 - Error Calling SQLite API 'sqlite3_mprintf'
	2 - Error converting string to UTF-8
	3 - Error reading escaped string

---  Примечания  ---
The escaped string is already wrapped with single quotes.
For example "It's a fine day" would look like "It''s a fine day"
For binary data use _SQLite_Encode()
