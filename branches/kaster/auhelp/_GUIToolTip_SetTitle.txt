
_GUIToolTip_SetTitle

Adds a standard icon and title string

#Include 
_GUIToolTip_SetTitle($hWnd, $sTitle[, $iIcon = 0])

---  Таблица параметров  ---
$hWnd
	Handle to control
$sTitle
	Title string
$iIcon
	[опционально] Set to one of the values below:.
		0 - No icon
		1 - Information icon
		2 - Warning icon
		3 - Error Icon

---  Таблица возвращаемых значений  ---
Успех:	True
Ошибка:	False

---  Примечания  ---
As of Windows XP SP2 and later, $iIcon can contain an HICON value. Any value greater than 3 is assumed to be
an HICON.
