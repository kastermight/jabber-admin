
_GUICtrlToolbar_GetBitmapFlags

Retrieves the flags that describe the type of bitmap to be used

#Include 
_GUICtrlToolbar_GetBitmapFlags($hWnd)

---  Таблица параметров  ---
$hWnd
	Handle to the control

---  Таблица возвращаемых значений  ---
Успех: A value that describes the type of bitmap that should be used:
		0 - Use small (16 x 16) bitmaps
		1 - Use large (24 x 24) bitmaps

---  Примечания  ---
The value returned is only advisory. The control recommends large or small bitmaps based upon whether the user
has chosen large or small fonts.
