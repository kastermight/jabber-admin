
_GUICtrlRichEdit_StreamToVar

Записывает содержание элемента Rich Edit в переменную

#Include 
_GUICtrlRichEdit_StreamToVar($hWnd [, $fRtf = True [, $fIncludeCOM=True [, $iOpts=0[, $iCodePage = 0]]]])

---  Таблица параметров  ---
$hWnd
	Дескриптор элемента
$fRtf
	[опционально] True - записывать Rich Text Format (RTF) (По умолчанию)
		False - записывать только текст
$fIncludeCOM
	[опционально] True (по умолчанию):
		    Если записывается RTF, то включает в себя любые COM-объекты (space consuming)
		    Если записывается только текст, то записывается текстовое представление COM-объектов
		False: Записывает пробелы вместо COM-объектов
$iOpts
	[опционально] additional options:
		$SFF_PLAINTRTF - записывать только rich text keywords общий для всех языков
		$SF_UNICODE - записывать Unicode
$iCodePage
	[опционально] Создать UTF-8 и текст использующий эту кодовую страницу
		По умолчанию: не использовать

---  Таблица возвращаемых значений  ---
Успех:	Возвращает RTF или текст
Ошибка:	Возвращает "" (пустую строку) и устанавливает @error
@error:	101 - $HWnd не является дескриптором
	1041 - $SFF_PLAINRTF недопустим для текстового файла
	1042 - $opts: недопустимые опции
	1043 - $SF_UNICODE допустимо только для текстовых файлов
	700 - внутренняя ошибка

---  Примечания  ---
Если текст выделен, то заменяется выделенное, иначе заменяется весь текст в элементе Rich Edit

Если расширение файла в $sFileSpec является .rtf, то записывается RTF, иначе текст

Используя функцию _GUICtrlRichEdit_IsModified(), чтобы определить, изменился ли текст
