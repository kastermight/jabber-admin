
_GUICtrlComboBox_SetHorizontalExtent

Set the width, in pixels, by which a list box can be scrolled horizontally

#Include 
_GUICtrlComboBox_SetHorizontalExtent($hWnd, $iWidth)

---  Таблица параметров  ---
$hWnd
	Handle to control
$iWidth
	Specifies the scrollable width of the list box, in pixels

---  Примечания  ---
If the width of the ListBox is smaller than $iWidth, the horizontal scroll bar horizontally scrolls
items in the list box.

If the width of the ListBox is equal to or greater than $iWidth, the horizontal scroll bar is hidden
or, if the ComboBox has the $CBS_DISABLENOSCROLL style, disabled.
