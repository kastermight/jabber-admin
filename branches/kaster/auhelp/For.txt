
For...To...Step...Next

Цикл, основанный на выражении (!!переписать!!).

For <variable> = <start> To <stop> [Step <stepval>]
	statements
	...
Next

---  Таблица параметров  ---
variable
	Переменная, используемая в качестве счётчика.
start
	Начальное значение переменной счётчика.
stop
	Конечное значение переменной счётчика.
stepval
	[опционально] Числовое значение (возможно, вещественное), на которое будет увеличено значение переменной счётчика при каждом шаге цикла. По умолчанию 1.

---  Примечания  ---
Переменная счётчика будет автоматически создана в локальной области видимости, даже если опция MustDeclareVars активна.

For...Next могут быть вложенными. Цикл For завершает свою работу, когда значение variable становится большим, чем значение stop.  Если stepVal или stop являются переменными, их значения устанавливаются только один раз при входе в цикл.

Цикл For выполняется ноль раз, если:
   start > stop и step > 0, или
   start < stop и step < 0
