
_ScreenCapture_CaptureWnd

Захватывает область экрана указанного окна

#Include 
_ScreenCapture_CaptureWnd($sFileName, $hWnd [, $iLeft = 0 [, $iTop = 0 [, $iRight = -1 [, $iBottom = -1 [, $fCursor = True]]]]])

---  Таблица параметров  ---
$sFileName
	Полный путь файла с расширением
$hWnd
	Дескриптор окна для захвата
$iLeft
	[опционально] X координата верхнего левого угла прямоугольника относительно окна.
$iTop
	[опционально] Y координата верхнего левого угла прямоугольника относительно окна.
$iRight
	[опционально] X координата правого нижнего угла прямоугольника относительно окна.
$iBottom
	[опционально] Y координата правого нижнего угла прямоугольника относительно окна.
$fCursor
	[опционально] Если True, то курсор тоже захватывается.

---  Примечания  ---
Если имя файла указано, то функция захватывает экран и сохраняет в файл. Если имя файла пустая строка, то функция захватывает экран и возвращает дескриптор HBITMAP растрового изображения. В этом случае, после завершения использования изображения необходимо удалить дескриптор вызовом функции _WinAPI_DeleteObject. Все координаты указываются относительно окна.

Требования GDI+: GDI+ необходимо распространять для приложений, которые запускаются в операционных системах Microsoft Windows NT 4.0 SP6, Windows 2000, Windows 98, и Windows Me.
