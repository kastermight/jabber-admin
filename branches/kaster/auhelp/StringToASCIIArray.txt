
StringToASCIIArray

Преобразует строку в массив, содержащий код ASCII каждого символа.

StringToASCIIArray ( "string", [start [, end [, encoding]]] )

---  Таблица параметров  ---
"string"
	Строка для преобразования в массив ASCII кодов.
start
	[опционально] позиция начала для обработки (По умолчанию: 0).
end
	[опционально] позиция конца для обработки (По умолчанию: StringLen("string")).
encoding
	[опционально] возвращаемый массив будет содержать значения в указанном наборе символов:
		0 - UTF-16 (По умолчанию)
		1 - ANSI
		2 - UTF-8

---  Таблица возвращаемых значений  ---
Успех:	Массив, где каждый элемент UNICODE код символа в соответствующей позиции.
Ошибка:	Возвращает пустую строку.

---  Примечания  ---
Строка может содержать символы Chr(0). Они будут появляться в возвращаемом массиве, а также любые данные, следующие после них. Обработка останавливается, только когда конец строки будет достигнут или до позиции указанной в параметре end.

Для того, чтобы преобразовать двоичные данные в массив с помощью этой функции, следует сначала преобразовать их в строку функцией BinaryToString().
