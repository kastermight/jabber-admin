
_Crypt_EncryptFile

Шифрует файл с указанным ключом и алгоритмом

#Include 
_Crypt_EncryptFile($sSourceFile, $sDestinationFile, $vCryptKey, $iALG_ID)

---  Таблица параметров  ---
$sSourceFile
	Файл для обработки
$sDestinationFile
	Файл для сохранения обработанного файла
$vCryptKey
	Пароль или указатель ключа, если указан флаг CALG_USERKEY
$iALG_ID
	Алгоритм шифрования

---  Таблица возвращаемых значений  ---
Успех:	Возвращает True
	Устанавливает @error = 0
Ошибка:	Возвращает -1 и устанавливает @error:
	1 - Не удалось создать ключ
	2 - Не удалось открыть файл  для обработки
	3 - Не удалось открыть файл для сохранения
	4 - Не удалось зашифровать заключительную часть
	5 - Не удалось зашифровать часть

---  Примечания  ---
Выходной файл может быть больше, чем входной файл в зависимости от алгоритма шифрования.
