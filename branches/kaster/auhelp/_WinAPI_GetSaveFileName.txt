
_WinAPI_GetSaveFileName

Creates a Save dialog box that lets the user specify the drive, directory, and name of a file to save

#Include 
_WinAPI_GetSaveFileName([$sTitle = "" [, $sFilter = "All files (*.*)" [, $sInitalDir = "." [, $sDefaultFile = "" [, $sDefaultExt = "" [, $iFilterIndex = 1 [, $iFlags = 0 [, $iFlagsEx = 0 [, $hwndOwner = 0]]]]]]]]])

---  Таблица параметров  ---
$sTitle
	[опционально] string to be placed in the title bar of the dialog box
$sFilter
	[опционально] Pairs of filter strings (for example "Text Files (*.txt)|All Files (*.*)")
		The first string in each pair is a display string that describes the filter (for example, "Text Files")
		The second string specifies the filter pattern (for example, "*.TXT")
		To specify multiple filter patterns for a single display string, use a semicolon to separate the patterns (for example, "*.TXT;*.DOC;*.BAK")
		A pattern string can be a combination of valid file name characters and the asterisk (*) wildcard character
		Do not include spaces in the pattern string.
$sInitalDir
	[опционально] String that can specify the initial directory
$sDefaultFile
	[опционально] A file name used to initialize the File Name edit control
$sDefaultExt
	[опционально] String that contains the default extension
$iFilterIndex
	[опционально] Specifies the index of the currently selected filter in the File Types control
$iFlags
	[опционально] See Flags in $tagOPENFILENAME information
$iFlagsEx
	[опционально] See FlagEx in $tagOPENFILENAME information
$hwndOwner
	[опционально] Handle to the window that owns the dialog box. This member can be any valid window handle, or it can be 0 if the dialog box has no owner

---  Таблица возвращаемых значений  ---
Успех:	Array in the following format:
	[0] - Contains the number of strings (2)
	[1] - Contains the path selected
	[2] - Contains file selected
Ошибка:	Array of 1 item set to 0
