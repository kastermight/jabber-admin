
_GUICtrlToolbar_GetButtonSize

Retrieves the current button width and height, in pixels

#Include 
_GUICtrlToolbar_GetButtonSize($hWnd)

---  Таблица параметров  ---
$hWnd
	Handle to the control

---  Таблица возвращаемых значений  ---
Успех: Array with the following format:
		[0] - Button height
		[1] - Button width
