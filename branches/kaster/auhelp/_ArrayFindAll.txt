
_ArrayFindAll

Найти индексы всех ячеек соответствующие поисковому запросу между двумя указанными ячейками одномерного или двумерного массива используя _ArraySearch() в цикле.

#Include 
_ArrayFindAll(Const ByRef $avArray, $vValue [, $iStart = 0 [, $iEnd = 0 [, $iCase = 0 [, $iPartial = 0 [, $iSubItem = 0]]]]])

---  Таблица параметров  ---
$avArray
	Массив для поиска
$vValue
	Значение для поиска в массиве
$iStart
	[опционально] Индекс массива, от которого начинать поиск
$iEnd
	[опционально] Индекс массива, в котором закончить поиск
$iCase
	[опционально] Если установлено в 1, поиск с учетом регистра (по умолчанию 0)
$iPartial
	[опционально] Если установлено в 1, осуществляется поиск части текста в содержимом каждой ячейки (по умолчанию 0)
$iSubItem
	[опционально] Индекс колонки для поиска в двумерном массиве

---  Таблица возвращаемых значений  ---
Успех:	Возвращает массив всех индексов, содержащих значение $vValue в массиве
Ошибка:	Возвращает -1, устанавливает @error (смотрите описание кодов ошибок функции _ArraySearch() )

---  Примечания  ---
Возвращаемый массив не содержит количество индексов в первом элементе, поэтому необходимо воспользоваться функцией UBound.
