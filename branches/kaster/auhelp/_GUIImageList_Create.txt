
_GUIImageList_Create

Создаёт элемент ImageList

#Include 
_GUIImageList_Create([$iCX=16[, $iCY=16[, $iColor=4[, $iOptions=0[, $iInitial=4[, $iGrow=4]]]]]])

---  Таблица параметров  ---
$iCX
	[опционально] Ширина каждого изображения в пикселях
$iCY
	[опционально] Высота каждого изображения в пикселях
$iColor
	[опционально] Глубина цвета изображения:
		0 - Use the default behavior
		1 - Use a 4 bit DIB section
		2 - Use a 8 bit DIB section
		3 - Use a 16 bit DIB section
		4 - Use a 24 bit DIB section
		5 - Use a 32 bit DIB section
		6 - Use a device-dependent bitmap
$iOptions
	[опционально] Option flags. Can be a combination of the following:
		1 - Use a mask
		2 - The images in the lists are mirrored
		4 - The image list contains a strip of images
$iInitial
	[опционально] Number of images that the image list initially contains
$iGrow
	[опционально] Number of images by which the image list can grow when the system needs to make room for new images. This parameter represents the number of new images that the resized image list can contain.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает дескриптор нового элемента
Ошибка:	Возвращает 0
