
_ScreenCapture_Capture

Захватывает область экрана

#Include 
_ScreenCapture_Capture([$sFileName = "" [, $iLeft = 0 [, $iTop = 0 [, $iRight = -1 [, $iBottom = -1 [, $fCursor = True]]]]]])

---  Таблица параметров  ---
$sFileName
	[опционально] Полный путь файла с расширением
$iLeft
	[опционально] X координата верхнего левого угла прямоугольника
$iTop
	[опционально] Y координата верхнего левого угла прямоугольника
$iRight
	[опционально] X координата правого нижнего угла прямоугольника. Если -1, то используется ширина текущего экрана.
$iBottom
	[опционально] Y координата правого нижнего угла прямоугольника. Если -1, то используется высота текущего экрана.
$fCursor
	[опционально] Если True, то курсор тоже захватывается.

---  Примечания  ---
Если имя файла указано, то функция захватывает экран и сохраняет в файл. Если имя файла пустая строка, то функция захватывает экран и возвращает дескриптор HBITMAP растрового изображения. В этом случае, после завершения использования изображения необходимо удалить дескриптор вызовом функции _WinAPI_DeleteObject.

Требования GDI+: GDI+ необходимо распространять для приложений, которые запускаются в операционных системах Microsoft Windows NT 4.0 SP6, Windows 2000, Windows 98, и Windows Me.
