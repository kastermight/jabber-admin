
GUICtrlCreateAvi

Создает элемент AVI видео для GUI.

GUICtrlCreateAvi ( filename, subfileid, left, top [, width [, height [, style [, exStyle]]]] )

---  Таблица параметров  ---
filename
	Имя видео файла. Поддерживаются только *.avi файлы.
subfileid
	Номер используемого ресурса. Если файл содержит только один видео ресурс, то использовать 0.
left
	Координата левого края. Если -1 тогда координата в соответствии с GUICoordMode.
top
	Координата верхнего края. Если -1 тогда координата в соответствии с GUICoordMode.
width
	[опционально] Ширина элемента (по умолчанию предыдущее значение).
height
	[опционально] Высота элемента (по умолчанию предыдущее значение).
style
	[опционально] Стиль элемента. Смотрите Стили элементов GUI в приложении.
	
	по умолчанию (-1) : $ACS_TRANSPARENT 
	$ACS_TRANSPARENT всегда используется, если не указан $ACS_NONTRANSPARENT.
exStyle
	[опционально] Расширенный стиль элемента. Смотрите таблицу расширенных стилей.

---  Таблица возвращаемых значений  ---
Успех:	Возвращает идентификатор (controlID) нового элемента.
Ошибка:	Возвращает 0.

---  Примечания  ---
Для получения параметров элемента смотрите GUICtrlRead.
Чтобы установить или изменить параметры элемента смотрите GUICtrlUpdate....

Чтобы воспроизвести видео, сразу при создании элемента, используйте стиль $ACS_AUTOPLAY.
Вы можете запускать и останавливать видео, установив состояние 1 или 0, используя GUICtrlSetState. Смотрите пример.

Чтобы объединить стили со стилем по умолчанию используйте BitOr($GUI_SS_DEFAULT_AVI, новый стиль,...).

Чтобы использовать параметры, указанные выше, вы должны включить в ваш скрипт #include <AVIConstants.au3>.

По умолчанию стиль изменения размеров $GUI_DOCKSIZE (не изменяется размер).
